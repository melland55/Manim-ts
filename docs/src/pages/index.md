# manim-ts

**Mathematical animations in the browser, powered by TypeScript.**

manim-ts is a faithful TypeScript port of [3Blue1Brown's Manim](https://github.com/3b1b/manim) that runs natively in web browsers.

## Features

- **Browser Native** -- No Python, no server. Runs entirely in the browser with Canvas2D rendering.
- **Mathematical Fidelity** -- Built on numpy-ts for exact mathematical parity with Python Manim's numpy operations.
- **Full Manim API** -- 500+ exported symbols covering mobjects, animations, scenes, cameras, and utilities.
- **TypeScript First** -- Full type safety, IDE autocompletion, and modern tooling.
